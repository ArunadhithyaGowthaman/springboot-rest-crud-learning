Project is done while learning Spring Boot fundamentals.

# Features
- RESTful CRUD APIs
-Layered Architecture (Controller, Service, Repository)
- Global Exception Handling using @ControllerAdvice
- Custom runtime exceptions
- Validation logic in service layer
- Tested with Postman
## Technologies Used
 Resource:
- JAVA
- Spring Boot
- Spring Data JPA
- H2 / MySQL
- Postman
## Learning Focus
* Separation of concerns
- REST API design
- Exception handling
- Dependency Injection(constructor injection for best practice)
